📋 Summary

Notion's homepage works well for existing users and organic traffic, but leaves room for improvement for new visitors who don't know the product yet. The page is built for brand awareness rather than getting new visitors to sign up. This analysis identifies 7 issues and suggests changes based on what tends to work well on similar B2B SaaS pages.

Key finding: Notion has 100M+ users but doesn't mention this anywhere on their homepage. Adding a visible user count is a simple change that would likely increase trust for new visitors — it's one of the first things I'd test.

🧪 Suggested A/B Tests

Three changes I'd test first, based on what's most clearly missing from the current page:

  • Headline: Change "Write. Plan. Collaborate." to something that includes Notion's user count — for example, "Used by 100M+ people. Get your whole team in one place." New visitors have more reason to sign up when they see a specific number.
  • Social proof above fold: Add "100M+ users" and a G2 review badge directly below the hero section. Currently no trust signals are visible without scrolling down significantly.
  • Navigation: Reduce nav links from 6 to 2 (Product + Pricing). More links = more ways to leave before signing up.

📈 GA4 Tracking Plan

To measure each test properly I'd set up custom events in GA4 via GTM: CTA button clicks, scroll depth milestones (25%, 50%, 75%), nav link clicks to see which links pull people away, and signup form start/complete events to track drop-off at the form step. With these in place, I could see exactly which part of the page visitors are leaving at — and whether the test changes are actually moving those numbers.

📅 How I'd Run the Tests

I'd run one test at a time so it's clear which change is making the difference. Start with the headline test (biggest potential impact), let it run for at least 3 weeks before looking at results, then move on to social proof and navigation. Before any tests start, I'd record a 2-week baseline in GA4 to have something to compare against.

💡 Key Takeaway

Notion has the strongest product and largest user base in its category, but its homepage doesn't reflect that. The page feels like a brand site rather than a page designed to get new visitors to sign up. The simplest fix is making their 100M+ user count visible — it's the strongest trust signal available, and it's completely missing from the page.
